# 📊 Sales Performance Analytics Dashboard


## 📌 Project Overview

Sales Performance Analytics Dashboard is a business intelligence web application designed to analyze sales data and generate meaningful business insights.

The dashboard helps organizations track sales performance, profit trends, customer activity, product performance, and regional sales using interactive visualizations.


---

# 🚀 Features


## 📈 Dashboard

- Total Sales KPI
- Total Profit KPI
- Total Orders
- Total Customers
- Best Selling Product
- Sales Visualization


## 📊 Analytics

- Product Performance Analysis
- Region Wise Sales Analysis
- Category Wise Sales Analysis
- Interactive Charts


## 📑 Reports

- Business Summary
- Sales Report
- Profit Report
- Customer Overview


## 📂 Dataset Upload

- Upload custom CSV sales datasets
- Automatic dashboard update
- CSV based analysis

# ⚙️ Installation & Setup


### 1. Clone Repository


```
git clone <repository-url>
```


### 2. Navigate Project Folder


```
cd Sales_Performance_Analytics_Dashboard
```


### 3. Install Requirements


```
pip install -r requirements.txt
```


### 4. Run Application


```
python app.py
```


### 5. Open Browser


```
http://127.0.0.1:5000
```
